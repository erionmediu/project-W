### commands 

    git status: Displays the current branch, staged changes, and untracked files.
    git add .: Stages all new or modified files for the next commit.
    git commit -m "Message": Saves staged changes with a descriptive message.
    git push origin main: Uploads commits to the main branch on the remote repository.
    git pull: Fetches and merges changes from the remote repository to your local branch.
    git clone [URL]: Creates a local copy of a remote repository.
    git branch: Lists all local branches in your repository.
    git checkout [branch-name]: Switches to the specified branch.
    git checkout -b [name] creates a new branch
    git merge [branch]: Merges the specified branch into the current branch.
    git log: Shows a history of commits for the current branch.
